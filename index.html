Overview

ModelNet-R refines the ModelNet40 dataset by resolving inconsistencies, removing 2D-like data, and improving size normalization.

Point-SkipNet is a lightweight yet powerful 3D classification model that enhances feature extraction using skip connections.

Video Presentation

Challenges in ModelNet40

  • Inconsistent labeling and ambiguous class definitions
  • Presence of nearly 2D objects lacking volumetric depth
  • Size mismatches causing classification confusion

Our Solution: ModelNet窶然 & Point窶全kipNet

We introduce:

  1. ModelNet窶然 Dataset: A refined dataset with corrected labels, removal of 2D artifacts, and enhanced class differentiation.
  2. Point窶全kipNet Architecture: A lightweight, graph窶礎ased neural network using sampling, grouping, and skip connections.

Dataset

Place these datasets in the `data` folder:

Training & Testing

Train Point-SkipNet from scratch:

cd Point-SkipNet
dataset="modelnetR" bash run_train.sh

Test with a trained checkpoint:

cd Point-SkipNet
dataset="modelnetR" bash run_test.sh

Experimental Results

Accuracy on datasets:

  • ModelNet: OA ~92.29%, mAcc ~89.84%
  • ModelNet-R: OA ~94.33%, mAcc ~92.93%
