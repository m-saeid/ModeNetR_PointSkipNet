Challenges in ModelNet40

ModelNet40 has long been a benchmark for 3D classification but faces issues:

  • Inconsistent labeling and ambiguous class definitions
  • Presence of nearly 2D objects lacking volumetric depth
  • Size mismatches causing classification confusion
  • Low differentiation between similar classes

Our Solution: ModelNet‑R & Point‑SkipNet

We tackle these issues with:

  1. ModelNet‑R Dataset: A refined version with improved labels, removal of 2D artifacts, and clearer class differentiation.
  2. Point‑SkipNet Architecture: A lightweight neural network using sampling, grouping, and skip connections for efficient classification.
