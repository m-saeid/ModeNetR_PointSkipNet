ModelNet‑R & Point‑SkipNet

Refining datasets. Enhancing architectures.

Overview

Welcome to our project, where we address the limitations of the traditional ModelNet40 dataset by introducing the refined ModelNet‑R and the innovative, lightweight Point‑SkipNet architecture. Our approach aims to deliver state‑of‑the‑art performance in 3D point cloud classification while keeping computational costs low.

Challenges in ModelNet40

ModelNet40 has been a cornerstone for 3D point cloud classification, yet it suffers from:

  • Inconsistent labeling and ambiguous classes
  • Presence of nearly 2D data lacking volumetric depth
  • Size mismatches that cause classification confusion
  • Poor differentiation between similar classes

Our Solution: ModelNet‑R & Point‑SkipNet

We tackle these challenges with a two‑fold solution:

  1. ModelNet‑R Dataset: A refined version of ModelNet40 with corrected labels, removal of 2D artifacts, adjusted size representations, and enhanced class differentiation.
  2. Point‑SkipNet Architecture: A lightweight, graph‑based neural network that employs efficient sampling, grouping, and skip connections to maintain high accuracy with reduced computation.

Experimental Insights

Our extensive experiments highlight significant improvements in classification accuracy when using the refined ModelNet‑R dataset combined with Point‑SkipNet.

  • Point‑SkipNet on ModelNet: Overall Accuracy ~92.29%, Mean Class Accuracy ~89.84%
  • Point‑SkipNet on ModelNet‑R: Overall Accuracy ~94.33%, Mean Class Accuracy ~92.93%

These metrics emphasize the vital role of dataset quality and model efficiency in advancing 3D point cloud classification.
